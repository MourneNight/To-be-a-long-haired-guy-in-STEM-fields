
#### 所需头文件
```c
#include <stdlib.h>  // rand(), srand()
#include <time.h>     // time()
```

#### 伪随机原理
- `rand()`：生成伪随机整数（0 ~ RAND_MAX）
- `srand(seed)`：设置随机种子，相同种子产生相同序列

#### 实现“更随机”效果
使用当前时间作为种子（时间不断变化）：
```c
srand(time(NULL));  // 只调用一次，通常放在 main 开头
```

#### 限制随机数范围
使用取模 `%` 和加法调整到指定区间 `[min, max]`：
```c
int num = rand() % n + min;  // 生成 [min, min+n-1]
```

**常见示例**：

| 需求               | 代码                                  | 范围         |
|--------------------|---------------------------------------|--------------|
| 1 ~ 100            | `rand() % 100 + 1`                    | 1 ~ 100     |
| 0 ~ 99             | `rand() % 100`                        | 0 ~ 99      |
| 10 ~ 50            | `rand() % 41 + 10`                    | 10 ~ 50     |
| 自定义 [a, b]      | `rand() % (b - a + 1) + a`            | a ~ b       |

#### 完整示例
```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));          // 初始化种子（只调用一次）

    int num = rand() % 100 + 1; // 生成 1~100 的随机数
    printf("随机数: %d\n", num);

    return 0;
}
```

**注意**：
- `srand(time(NULL))` 只需调用一次（多次调用可能因时间未变而重复种子）
- `RAND_MAX` 通常为 32767（短整型）或更大，视平台而定
- 若需更高随机质量，可考虑现代 C 使用 `<random>`（C23）或第三方库
