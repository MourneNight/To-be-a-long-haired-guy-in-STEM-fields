***所需头文件***
```c
#include<stdlib.h> //调用 rand 和 srand 函数
#include<time.h>  //调用 time 函数
```
***伪随机的达成***
**rand()**
是伪随机
通过将种子进行一系列计算，获取一个伪随机的数列
**srand()**
参数为种子
该值会被带入 rand 中进行计算

***如何达成真正的随机？***
**time()**
参数是指针，函数会将时间存储在该变量中，若不需要则输入NULL
time返回Unix纪元，时间不断变化，从而是的srand获得的种子处于变化中，从而达到随机效果
```c
srand(time(NULL));
```
借助此行代码，获得一个随机数
**对随机数范围的限制：**
通过取模运算和加法，我们可以限制随机数范围
```c
srand(time(NULL));
int random_number = rand() % 100 + 1;
```
此代码便可将随机数限制在1~100之间
